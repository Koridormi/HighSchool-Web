# Proyecto Web - Colegio Secundario Solaris

Sitio Web de un Colegio (Ficticio) con 5 secciones, responsive, un formulario con validacion JS.

## Demo

* Sitio en vivo: [Ver proyecto](https://solaris-school.netlify.app/)
